## 显示图像
在前面结尾部分，通过在 `surface` 表现画一个实心矩形来检测是否生效，接下来可以用更通用的方案，把图像显示在 `surface` 表面

``` c
SDL_Surface *image = SDL_LoadBMP("bmp_24.bmp");
if (image == NULL) {
    SDL_Log("Error loading image: %s\n", SDL_GetError());
    return 1;
}
```
先用 `SDL_LoadBMP` 加载 `bmp` 格式的图片，如果加载报错就输出对应信息并且退出程序

``` c
Uint32 color = SDL_MapSurfaceRGB(surface, 0, 0, 0);
SDL_FillSurfaceRect(surface, NULL, color);
SDL_BlitSurface(image, NULL, surface, NULL);
SDL_UpdateWindowSurface(window);
```
先用黑色填满 `surface`，再把 `image` 的像素复制到 `surface`，`image` 的左上角和 `surface` 的左上角对齐，都在 `(0, 0)` 的位置。
- 如果 `image` 比 `surface` 小，那么 `surface` 的其他区域会维持不变，也就是继续保持为填充色黑色。
- 如果 `image` 比 `surface` 大，那么会对 `image` 进行裁剪，只显示 `surface` 那么大的区域

`SDL_BlitSurface` 第二个参数是指显示 `image` 图片的尺寸，第四个参数是指 `image` 图片显示在 `surface` 什么位置

下面的代码是指从 `image` 图片 `(100, 60)` 的位置裁剪出 `90 * 40` 的区域，然后显示在 `surface` 表面 `(50, 150)` 的位置，这里需要注意，尽管 rect 指定了 `w` 与 `h`，但并不起作用。如果想实现缩放，需要使用 `SDL_BlitSurfaceScaled` 函数，它会根据 rect2 的 `w` 和 `h` 进行缩放
``` c
Uint32 color = SDL_MapSurfaceRGB(surface, 0, 0, 0);
SDL_FillSurfaceRect(surface, NULL, color);

SDL_Surface *image = SDL_LoadBMP("bmp_24.bmp");
SDL_Rect rect1 = {
    .x = 100,
    .y = 60,
    .w = 90,
    .h = 40,
};
SDL_Rect rect2 = {
    .x = 50,
    .y = 150,
    .w = image->w,
    .h = image->h,
};
SDL_BlitSurface(image, &rect1, surface, &rect2);
SDL_UpdateWindowSurface(window);
```

在 `SDL2` 里面画 `png` 图片比较麻烦，需要额外使用 `texture` 和 `renderer` 的概念，但是在 `SDL3` 中，画 `png` 和画 `bmp` 一样方便
``` c
SDL_Surface *image = SDL_LoadPNG("bmp_24.png");
```
其他地方保持不变，这样就可以从画 `bmp` 变成画 `png`

``` c
SDL_DestroySurface(image);
SDL_DestroyWindow(window);
SDL_Quit();
```
最后释放 `image` 和 `window` 资源
