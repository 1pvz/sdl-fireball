## 整理代码结构
### 拆分 Paddle
目前实现了正确画图片的功能，并且绑定了事件，按键之后图片可以流畅移动

但是现在的程序所有功能写在 `main` 函数里面，并不利于接下来添加新代码，可维护性太低，所以先把现有代码结构整理下

先把核心 `Paddle` 抽出来，定义 `Paddle` 的数据类型，有这些属性

``` c
struct _Paddle {
    SDL_Surface *image;
    int x;
    int y;
    int speed;
};
typedef struct _Paddle Paddle;
```

然后创建 `Paddle_new` 函数用来创建 `paddle`，因为 `paddle` 会持有 `image` 属性，并且加载图片是通用需求，所以单独抽取 `imageFromPath` 函数
``` c
SDL_Surface *
imageFromPath(const char *path) {
    SDL_Surface *image = SDL_LoadPNG(path);
    if (image == NULL) {
        SDL_Log("load image error %s\n", SDL_GetError());
    }
    return image;
}

Paddle *
Paddle_new(void) {
    SDL_Surface *image = imageFromPath("paddle.png");
    Paddle *o = malloc(sizeof(Paddle));
    o->image = image;
    o->x = 100;
    o->y = 250;
    o->speed = 15;
    return o;
}
```

提取 `paddle` 并且把 `x` `y` `speed` `image` 这些属性放进去之后，那原本用到这些变量的地方全都需要改成 `paddle` 相关版本

使用旧变量的地方依次为

``` c
int speed = 5;
SDL_Surface *image = SDL_LoadPNG("paddle.png");
SDL_Rect rect = {
    .x = 50,
    .y = 150,
    .w = image->w,
    .h = image->h,
};

if (keyA_pressed) {
    SDL_Log("key a pressed");
    rect.x -= paddle->speed;
}
if (keyD_pressed) {
    SDL_Log("key d pressed");
    rect.x += paddle->speed;
}

SDL_BlitSurface(image, NULL, surface, &rect);

SDL_DestroySurface(image);
```

这些地方都需要改成 `paddle`

``` c
Paddle *paddle = Paddle_new();
SDL_Rect rect = {
    .x = paddle->x,
    .y = paddle->y,
    .w = paddle->image->w,
    .h = paddle->image->h,
};

if (keyA_pressed) {
    SDL_Log("key a pressed");
    rect.x -= paddle->speed;
}
if (keyD_pressed) {
    SDL_Log("key d pressed");
    rect.x += paddle->speed;
}

SDL_BlitSurface(paddle->image, NULL, surface, &rect);

SDL_DestroySurface(paddle->image);
free(paddle);
```

需要特别注意的是，`paddle` 是通过 `malloc` 分配在堆上的变量，那在不使用的时候需要通过 `free` 来释放对应的内存

因为栈上分配的变量在离开作用域的时候会自动释放，而堆上分配的变量不会自动释放，所以要手动释放

观察可以发现 `rect` 变量实际上只在 `SDL_BlitSurface` 参数里使用，并且 `rect` 的所有属性都是和 `paddle` 相关，这种情况下移动的逻辑可以直接消除 `rect`

``` c
if (keyA_pressed) {
    SDL_Log("key a pressed");
    paddle->x -= paddle->speed;
}
if (keyD_pressed) {
    SDL_Log("key d pressed");
    paddle->x += paddle->speed;
}
```

在画 `paddle` 的时候临时创建 `rect` 变量并且把坐标 `(x, y)` 传过去，当然也可以省略这个变量直接写在参数里，但没必要
``` c
SDL_Rect rect = {
    .x = paddle->x,
    .y = paddle->y,
    .w = paddle->image->w,
    .h = paddle->image->h,
};
SDL_BlitSurface(paddle->image, NULL, surface, &rect);
```

考虑 `paddle` 的移动逻辑，既然是属于 `paddle` 的逻辑，那么也应该抽到 `paddle` 内部处理

``` c
if (keyA_pressed) {
    SDL_Log("key a pressed");
    paddle->x -= paddle->speed;
}
```

当然 C 语言实现函数会比较麻烦点，需要处理成函数指针的形式

所以先来修改 `Paddle` 结构体，给 `Paddle` 加上 `moveLeft` 和 `moveRight` 两个函数指针

``` c
struct _Paddle {
    SDL_Surface *image;
    int x;
    int y;
    int speed;
    void (*moveLeft)(struct _Paddle *paddle);
    void (*moveRight)(struct _Paddle *paddle);
};
typedef struct _Paddle Paddle;
```

接着定义这两个函数

``` c
void
Paddle_moveLeft(Paddle *self) {
    self->x -= self->speed;
}

void
Paddle_moveRight(Paddle *self) {
    self->x += self->speed;
}
```

这样就可以在 `Paddle_new` 函数里面补上函数指针的值

``` c
o->moveLeft = Paddle_moveLeft;
o->moveRight = Paddle_moveRight;
```

对应地，原本直接修改 `paddle` 坐标的地方就可以通过调用函数来完成了

``` c
if (keyA_pressed) {
    paddle->moveLeft(paddle);
}
if (keyD_pressed) {
    paddle->moveRight(paddle);
}
```

### 拆分 Game
截止到目前为止，`Paddle` 部分已经处理好了，接下来需要处理剩余的部分，仿照 `Paddle` 的逻辑来处理 `Game`

先让 `Game` 持有 `window` 和 `surface` 两个属性
``` c
struct _Game {
    SDL_Window *window;
    SDL_Surface *surface;
};
typedef struct _Game Game;
```

这样就可以创建 `game` 并且把 `window` 和 `surface` 放进去
``` c
Game *
Game_new(void) {
    Game *g = malloc(sizeof(Game));
    const int w = 640;
    const int h = 480;
    SDL_Window *window = SDL_CreateWindow("Hello SDL3", w, h, 0);
    if (window == NULL) {
        SDL_Log("sdl create window failed %s\n", SDL_GetError());
        return NULL;
    }
    g->window = window;

    SDL_Surface *surface = SDL_GetWindowSurface(window);
    if (surface == NULL) {
        SDL_Log("Error getting surface: %s\n", SDL_GetError());
        return NULL;
    }
    g->surface = surface;

    return g;
}
```

然后依次把用到 `window` 和 `surface` 的地方改成 `game` 持有的版本

当然也不能忘记手动调用 `free` 函数释放资源

``` c
Game *game = Game_new();
if (game == NULL) {
    SDL_Quit();
    return 1;
}

Uint32 color = SDL_MapSurfaceRGB(game->surface, 255, 255, 255);
SDL_FillSurfaceRect(game->surface, NULL, color);
SDL_Rect rect = {
    .x = paddle->x,
    .y = paddle->y,
    .w = paddle->image->w,
    .h = paddle->image->h,
};
SDL_BlitSurface(paddle->image, NULL, game->surface, &rect);
SDL_UpdateWindowSurface(game->window);

SDL_DestroySurface(paddle->image);
free(paddle);
SDL_DestroyWindow(game->window);
free(game);
```
