## debug 工具
### 暂停功能
程序很难一次性写对，所以方便的调试方案很重要

对于游戏来说，始终会运行一个游戏循环，这点与 Web 程序很不一样。假设 `ball` 与 `block` 碰撞后，`block` 没有消失（如果碰撞函数实现不对，这很容易发生），那调试就会比较麻烦，最好是希望能 “抓住” 那一帧的状态

这意味着程序要有暂停功能，暂停之后，场景里的元素都不运动，这样就方便调试，因为目前没有音效和背景音乐的需求，所以可以把整个游戏直接暂停

停掉游戏的逻辑很简单，现在所有逻辑都放在 `Game_update` 里完成，所以直接中断这个函数的执行

先给 `game` 增加属性 `paused`，创建的时候初始化为 `false`

``` c
// game.h
struct _Game {
    // 其他属性
    bool paused;
};
```

`Game_update` 里面判断 `paused` 如果为 `true` 就不让函数继续执行

``` c
// game.c
void
Game_update(Game *self) {
    Game *game = self;
    if (game->paused) {
        return;
    }
    Ball_move(game->ball);
    if (Paddle_collide(game->paddle, game->ball->image)) {
        Ball_bounce(game->ball);
    }
    for (int i = 0; i < game->numberOfBlocks; i += 1) {
        Block *block = game->blocks[i];
        if (Block_collide(block, game->ball->image)) {
            Block_kill(block);
            Ball_bounce(game->ball);
        }
    }
}
```

接下来就是处理如何改变 `paused` 的值，最方便的方案是直接绑定按键，比如说按了 `p` 键就暂停，再次按 `p` 就恢复游戏，也就是每次按下 `p` 都会对 `paused` 进行取反操作

在 `SDL_EVENT_KEY_DOWN` 事件中补充这个判断

``` c
if (event.type == SDL_EVENT_KEY_DOWN) {
    SDL_Scancode sc = event.key.scancode;
    if (sc < MAX_COUNT) {
        game->keydowns[sc] = true;
    }
    if (sc == SDL_SCANCODE_P) {
        game->paused = !game->paused;
    }
}
```

### 拖拽

暂停是为了方便拿到状态，最好还能直接复现状态，比如让 `ball` 撞击 `block`，但 `ball` 的运行路线我们很难控制，自然就不容易出现预期的碰撞行为

最好是手动把 `ball` 放在特定位置，比如就放在 `block` 旁边，那就很容易发生碰撞了

虽然可以在代码里面直接指定 `ball` 的位置，但需要反复尝试比较麻烦，所以想直接拖动小球到特定为止

拖动的逻辑是按住鼠标左键不松开，拖到哪里 `ball` 就跟到哪里

在 `game` 里面增加 `mouseDown` 用来记录是否按下鼠标，初始值为 `false`

``` c
// game.h
struct _Game {
    // 其他属性
    bool mouseDown;
};
```

监听多个事件
- `SDL_EVENT_MOUSE_BUTTON_DOWN` 鼠标按下的时候，`mouseDown` 设置为 `true`
- `SDL_EVENT_MOUSE_BUTTON_UP` 鼠标抬起的时候，`mouseDown` 设置为 `false`
- `SDL_EVENT_MOUSE_MOTION` 鼠标移动的时候，如何 `mouseDown` 为 `true`，说明此时鼠标依然保持按下的状态，也就是按住并且移动，这个时候判断如果鼠标选中了 `ball` 就更新 `ball` 的坐标

``` c
void
Game_bindEvents(Game *self) {
    Game *game = self;
    SDL_Event event;
    while (SDL_PollEvent(&event)) {
        if (event.type == SDL_EVENT_QUIT) {
            game->quit = true;
        } else if (event.type == SDL_EVENT_KEY_DOWN) {
            SDL_Scancode sc = event.key.scancode;
            if (sc < MAX_COUNT) {
                game->keydowns[sc] = true;
            }
            if (sc == SDL_SCANCODE_P) {
                game->paused = !game->paused;
            }
        } else if (event.type == SDL_EVENT_KEY_UP) {
            SDL_Scancode sc = event.key.scancode;
            if (sc < MAX_COUNT) {
                game->keydowns[sc] = false;
            }
        } else if (event.type == SDL_EVENT_MOUSE_BUTTON_DOWN) {
            game->mouseDown = true;
        } else if (event.type == SDL_EVENT_MOUSE_BUTTON_UP) {
            game->mouseDown = false;
        } else if (event.type == SDL_EVENT_MOUSE_MOTION) {
            if (game->mouseDown) {
                float x = event.motion.x;
                float y = event.motion.y;
                SDL_Point point = {
                    .x = x,
                    .y = y,
                };
                SDL_Rect rect = {
                    .x = game->ball->image.x,
                    .y = game->ball->image.y,
                    .w = game->ball->image.image->w,
                    .h = game->ball->image.image->h,
                };
                if (SDL_PointInRect(&point, &rect)) {
                    SDL_Log("x and y is %f %f\n", x, y);
                    game->ball->image.x = x;
                    game->ball->image.y = y;
                }
            }
        }
    }
}
```
