## 添加 block
有 `paddle` 和 `ball` 之后，还需要补充 `block`，`block` 不会移动，被击中之后会消失

``` c
struct _Block {
    GuaImage image;
    bool alive;
};
typedef struct _Block Block;

Block *
Block_new(void) {
    SDL_Surface *image = imageFromPath("block.png");
    if (image == NULL) {
        return NULL;
    }
    Block *o = malloc(sizeof(Block));
    if (o == NULL) {
        SDL_DestroySurface(image);
        return NULL;
    }
    o->image.image = image;
    o->image.x = 100;
    o->image.y = 100;
    o->alive = true;
    return o;
}

void
Block_kill(Block *self) {
    self->alive = false;
}
```

`main` 函数里面创建 `block`，并且最后释放资源

``` c
Block *block = Block_new();
if (ball == NULL) {
    SDL_Log("Failed to create ball\n");
    SDL_DestroyWindow(game->window);
    free(game);
    SDL_Quit();
    return 1;
}
game->block = block;
// ...
// 游戏主循环
// ...
SDL_DestroySurface(block->image.image);
free(block);
```

`Game` 自然需要拿到 `Block`，也就是 `Game` 的 `struct` 里面需要补充 `block` 属性，初始化的时候设置为 `NULL`，`main` 函数里面创建 `block` 之后就挂在 `game` 上

这样才可以在 `Game_draw` 函数里面画出 `block`

``` c
if (block->alive) {
    Game_drawImage(game, block->image);
}
```

`Game_update` 里面还需要判断碰撞，碰撞之后调用 `Block_kill` 把 `block` 移除
``` c
void
Game_update(Game *self) {
    Game *game = self;
    Ball_move(game->ball);
    if (collide(game->paddle->image, game->ball->image)) {
        Ball_bounce(game->ball);
    }
    if (collide(game->block->image, game->ball->image)) {
        Block_kill(game->block);
    }
}
```
